I had coolsculpting done several months ago on my muffin top and inner thighs. I was very pleased with the results. I started to notice changes around a month after treatment. It was around that time that friends, family, and coworkers began asking me if I had lost weight. I hadn't told anyone I was having this done so I could get objective comments. I've actually put on some weight and still look dramatically better then I did before. I would highly recommend this!
